# Algorithm Test

## Run With Docker
- Install [Docker](https://www.docker.com/)
- Run the app

  `./app`

## Run Without Docker
- Install [Golang](https://golang.org/dl/)
- Build the app

  `go build`

- Run the app

  `./algorithm-test`

- Run in development

  `go run .`

## Test
- Run the tests

  `go test -v`
